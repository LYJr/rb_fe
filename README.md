### 토끼굴 프로젝트

## 개요
# 프로젝트 목표
개발 하면서 필요한 명령어나 키워드를 모아둔 개인 사이트 (예 : HTML 태그, 리눅스 명령어 등)

# 토끼굴 의미
토끼굴은 안을 볼 수 없기 때문에 어떤것이 들어있는지, 얼마나 큰지 알 수 없다는 것을 반대로 생각하여 "모든걸 수용할 수 있다" 라는 의미로 사용.

# 프로젝트 진행하며 익히고 싶은 것
1. css를 좀 더 잘 다룰 수 있게 되는 것.
2. 리액트 라우터를 좀 더 익숙해지는 것.
3. CRUD를 익숙하게 구현하는 것.
